# README
Space Station Manager is a Rails CRUD web app with hand-rolled API routes, a PostgreSQL database and feature tested with Capybara. 

## Local Setup
- `git clone` this repo
- `cd space_station_manager`
- `bundle`
- `rails db:{create, migrate}`
